# Stroke Prediction Analysis Portfolio

## Project Overview
This project analyzes healthcare data to predict stroke events and identify key risk factors. It includes data preprocessing, exploratory data analysis (EDA), and machine learning insights.

## Files
- **`index.html`**: The main portfolio report. When hosted on GitHub Pages, this serves as the landing page.
- **`Stroke_Prediction_Portfolio_PDF.html`**: A landscape-formatted version optimized for "Print to PDF" to create a slide-deck style report.
- **`Stroke_Prediction_Analysis.ipynb`**: The original Jupyter Notebook containing the full code and analysis.
- **`images/`**: Contains generated visualization plots used in the report.

## How to View
1.  **Online**: Enable [GitHub Pages](https://pages.github.com/) in your repository settings and point it to the `main` or `master` branch. The portfolio will be available at `https://[your-username].github.io/[repo-name]/`.
2.  **Local**: Open `index.html` in any web browser.

## PDF Generation
To generate a PDF report:
1.  Open `Stroke_Prediction_Portfolio_PDF.html` in your browser.
2.  Press `Cmd + P` (or `Ctrl + P`).
3.  Select "Save as PDF".
4.  Set Layout to **Landscape**.
5.  **Important**: Enable "Background graphics" in settings.
